# Evaluation Notes

EDUVSUM and older public benchmark experiments are retained only as historical or sanity-check material. The final project does not use them to claim summarization superiority.

What remains useful:

- real educational videos and subtitles can be used as workflow inputs;
- annotations can help inspect whether selected content overlaps human-marked important regions;
- batch runs are useful for engineering sanity checks.

What is not claimed:

- no public-method leadership comparison;
- no stable baseline superiority;
- no proof that the selector reliably finds the most important course segments;
- no industrial-scale benchmark conclusion.

The final project evidence is about workflow behavior: traceability, repair, gating, export validation, and human review readiness.
